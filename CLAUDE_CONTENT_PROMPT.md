# Claude Prompt Template for MIKATA Content

コピペ用の**全文テンプレート**（Role / Guardrails / MDX frontmatter / X 投稿など）は次の正本を開いてください。

→ **[web/docs/CLAUDE_CONTENT_PROMPT.md](web/docs/CLAUDE_CONTENT_PROMPT.md)**

## 実装と揃えておく場所

- 記事ファイル: `web/content/blog/{slug}.mdx`
- 公開 URL: `/blog/{slug}`（`canonical` もこの形式）
- プロダクト憲法: リポジトリ直下の `AGENTS.md`

テンプレを更新するときは **`web/docs/CLAUDE_CONTENT_PROMPT.md` のみ**編集すれば足ります。
