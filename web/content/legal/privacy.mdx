MIKATA（運営者：磯田龍太郎　以下「当方」）は、本プライバシーポリシー（以下「本ポリシー」）において、MIKATAサービス（以下「本サービス」）における個人情報の取り扱いを定めます。本サービスをご利用の前に必ずお読みください。

## 第1条（収集する情報）
当方は、以下の情報を収集します。
### 1. ユーザーが直接提供する情報
- アカウント登録情報（メールアドレス、パスワードのハッシュ値）
- 任意で提供されるプロフィール情報（年齢レンジ、職種タグ等）
- 本サービス利用時に入力するテキスト（以下「入力コンテンツ」）

### 2. 自動的に収集される情報
- アクセスログ（IPアドレス、ブラウザ種別、アクセス日時）
- 利用状況データ（機能使用回数、利用シーン選択等）
- デバイス情報（OS、端末識別子）

## 第2条（情報の利用目的）
収集した情報は以下の目的で利用します。
- 本サービスの提供・運営・改善
- AIによる回答生成（入力コンテンツはOpenAI, LLCのAPIに送信されます。詳細は第4条参照）
- ユーザーサポートへの対応
- 利用規約違反・不正利用の防止
- サービスの利用統計の作成（個人を特定しない形で実施）


## 第3条（情報の保存・保持期間・削除）
当方は、以下の方針で情報を保管・削除します。

保持期間
- アカウント情報：退会手続き完了後30日以内に削除
- 入力コンテンツおよびAI生成結果：生成後__日間保存（有料プランの履歴機能利用時は__日間）
- アクセスログ：90日間保存後に削除

削除手続き
退会またはデータ削除依頼を受けた場合、当方は受付から30日以内に以下のデータを削除します。
- Firestoreに保存された入力コンテンツおよびAI生成結果
- アカウント情報（メールアドレス等）
- 任意プロフィール情報

削除依頼の受付窓口：
① アプリ内「設定 > アカウント > アカウントを削除する」（即時申請可）
② メール：ryu346475@gmail.com（件名：「データ削除依頼」）
※ アクセスログは保持期間（90日）経過後に自動削除されます。

本サービスはAI生成機能のためにOpenAI, LLC（米国）のAPIを利用しています。

送信されるデータ
- ユーザーが入力したテキスト（入力コンテンツ）
- 生成に必要なシステムプロンプト


OpenAIのデータ取り扱い（確認済み事項）
以下はOpenAI APIデータポリシーに基づく2026年4月時点の確認内容です。
ポリシー変更時は本ポリシーを更新します。

④ 暗号化：通信はTLS 1.2以上で暗号化されます。


## 第5条（第三者提供）
当方は、以下の場合を除き、収集した個人情報を第三者に提供しません。
- ユーザー本人の同意がある場合
- 法令に基づく開示要請がある場合
- 人の生命・身体・財産の保護のために必要な場合
- 業務委託先への提供（下記参照、秘密保持義務を課します）

| サービス名 | 提供者・所在国 | 用途 |
| --- | --- | --- |
| OpenAI API | OpenAI, LLC（米国） | AI回答生成 |
| Firebase / Firestore | Google LLC（米国） | 認証・データ保存 |
| Stripe | Stripe, Inc.（米国） | 決済処理（課金実装後） |
| Vercel | Vercel, Inc.（米国） | Webホスティング |

## 第6条（機密情報・第三者情報の入力について）
本サービスへの入力内容はOpenAI（米国）のサーバーに送信されます。
ユーザーが自らの判断でお勤め先の機密情報または第三者の個人情報を入力した場合、それにより生じた一切の損害・紛争（お勤め先との紛争、顧客情報の漏えい等を含む）の責任はユーザー自身に帰属します。当方はこれについて一切の責任を負いません。

## 第7条（セキュリティ）
- 通信はHTTPS（TLS 1.2以上）で暗号化
- Firebase Security Rulesにより、ユーザーは自身のデータのみアクセス可能
- 本番環境のサーバーログに入力コンテンツの全文を出力しない
- アクセス権限は最小権限の原則で管理

## 第8条（ユーザーの権利）
ユーザーは以下の権利を行使できます。
- 個人情報の開示・訂正・追加・削除の請求
- 利用停止・消去の請求
- アカウント削除（アプリ内設定から実施。削除後30日以内に関連データを削除）
請求先：ryu346475@gmail.com
