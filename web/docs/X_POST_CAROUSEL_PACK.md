## 3) デザインの最小ルール（埋もれない用）

- 文字サイズ: タイトルは**太字・大きめ**（最低 72px 相当）

## 6) 実スクショ前提の制作手順（この順でやる）

### 6.1 先に用意するもの

- ブラウザで `https://bemikata.com/app?lp_variant=x_post` を開く
- 表示倍率: `100%`
- テーマ: ダーク推奨（統一感のため）
- 画面幅: `1440px` 前後（PC撮影なら）
- 個人情報は必ず伏せ字（A社、上司A、○万円）

### 6.2 撮るスクショ（最低3枚）

1. **入力画面（空欄〜タグあり）**  
   - シーン選択が見える  
   - クイックタグが2つ以上見える  
   - テキスト入力欄が見える

2. **生成中画面（進捗バー）**  
   - 「AIがカンペを生成中です…」が見える  
   - 進捗バーと経過秒が見える  
   - 「動いてる」証拠になる

3. **生成結果画面（価値が伝わる）**  
   - 「今の状況」「明日やること」「このまま送れる報告文」が見える  
   - Copyボタンが見える

### 6.3 画像加工ルール（Figma/Canva共通）

- キャンバス: `1200x1500`（4:5）
- 四辺の余白: 80px 以上
- 1枚1メッセージ厳守
- スクショ上に被せる文字は最大2ブロック
- 赤枠・矢印は1枚に1〜2個まで（多すぎると安っぽく見える）
